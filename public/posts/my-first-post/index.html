Кратчайший путь на сфере

Введение

Нахождение кратчайшего пути на плоской Земле - тривиальная задача. Соедините две точки прямой линией и готово. А вот решение подобной задачи на шарообразной Земле куда менее тривиально. Текущий материал, предназначенный для школьников, изучающих курс географии 6 класса, используя лишь методы элементарной математики покажет, по какой траектории двигаться, чтобы минимизировать расстояние между точками

Естественным будет начать, вспомнив основы дифференциальной геометрии. Локально минимизирует расстояние между двумя точками кривая, называемая геодезической. Важно, что между двумя точками может проходить не одна геодезическая, а также конкретная геодезическая может и не быть глобальным минимумом длины. Геодезическая, которая среди всех, проходящих через некоторые две точки, имеет минимальную длину, называется минимальной. Минимальные геодезические уже являются глобальными минимумами длины. Но в нашей задаче из сферической симметрии нашей поверхности следует то, что через две точки проходит единственная, полностью продолженная, геодезическая. А значит дуга между этими точками является минимумом расстояния.

Осталось лишь получить и решить уравнение геодезической.

Уравнение геодезической

Общий вид уравнения геодезической в координатах, предполагая соглашение Эйншетейна о суммировании: $$ \frac{\partial^{2} x^{k}}{\partial t^{2}} + \Gamma^{k} _{ij}\frac{\partial x^{i}}{∂ t}\frac{\partial x^{j}}{\partial t} = 0 $$

$\Gamma^{k} _{ij}$ - символы Кристоффеля второго рода. Введём систему координат и выразим в ней символы Кристоффеля. Тогда и получим искомое уравнение

Естественным выбором будет система сферических координат:

$$ \overrightarrow{r} = (R\cos{\theta}\cos{\phi}, R\cos{\theta}\sin{\phi}, R\sin{\theta})$$

В качестве базиса касательного пространства возьмём голономный, т.е. из частных производных по координатам.

$$ \overrightarrow{e _i} = \frac{\partial \overrightarrow{r}}{\partial x^{i}}, \overrightarrow{e _{\theta}} = \frac{\partial \overrightarrow{r}}{\partial \theta}, \overrightarrow{e _{\phi}} = \frac{\partial \overrightarrow{r}}{\partial \phi}$$

Тогда коммутаторы всех базисных векторов будут нулями, а значит символы Кристоффеля:

$$ \Gamma^{k} _{ij} = \frac{1}{2}g^{kl}\left(\frac{\partial g _{il}}{\partial x^{j}} + \frac{\partial g _{jl}}{\partial x^{i}} - \frac{\partial g _{ij}}{\partial x^{l}}\right) $$

$g _{ij}$ - метрика, а $g^{ij}$ - обратная метрика

Найдём сами базисные векторы, а также их скалярные произведения - компоненты метрики

$$\overrightarrow{e _{\theta}} = (-R\sin{\theta}\cos{\phi}, -R\sin{\theta}\sin{\phi}, R\cos{\theta})$$ $$\overrightarrow{e _{\phi}} = (-R\cos{\theta}\sin{\phi}, R\cos{\theta}\cos{\phi}, 0)$$ $$g _{ij} = \lang \overrightarrow{e _i}, \overrightarrow{e _j} \rang$$ $$g _{11} = \lang \overrightarrow{e _\phi}, \overrightarrow{e _\phi} \rang = (-R\cos{\theta}\sin{\phi})^{2} + (R\cos{\theta}\cos{\phi})^{2} = R^{2}\cos^2{\theta}$$ $$g _{22} = \lang \overrightarrow{e _\theta}, \overrightarrow{e _\theta} \rang = (-R\sin{\theta}\cos{\phi})^2 + (-R\sin{\theta}\sin{\phi})^2 + (R\cos{\theta})^2 = R^{2}$$ $$g _{12} = g _{21} = \lang \overrightarrow{e _\phi}, \overrightarrow{e _\theta} \rang = (-R\cos{\theta}\sin{\phi})\cdot(-R\sin{\theta}\cos{\phi}) + (R\cos{\theta}\cos{\phi})\cdot(-R\sin{\theta}\sin{\phi}) + 0 = 0$$ $$g _{ij} = \begin{pmatrix} R^{2}\cos^2{\theta} & 0 \\ 0 & R^{2} \end{pmatrix}$$

Обратную метрику в таком случае найти крайне просто. Обратная матрица к диагональной - диагональная, элементы которой равны обратным к изначальной

$$g^{ij} = \begin{pmatrix} \frac{1}{R^{2}\cos^2{\theta}} & 0 \\ 0 & \frac{1}{R^{2}} \end{pmatrix}$$

Воспользовавшись выражением для символов Кристоффеля получим, что не равны нулю будут только следующие:

$$ \Gamma^{2} _{11} = -\sin{\theta}\cos{\theta} $$ $$\Gamma^{1} _{12} = \Gamma^{1} _{21} = -\tan{\theta}$$

Отсюда получим уравнения геодезической в координатах $\theta$ и $\phi$

$$\begin{cases}\frac{\partial^{2} \theta}{\partial t^{2}} = \sin{\theta}\cos{\theta}\frac{\partial \phi}{∂ t}\frac{\partial \phi}{\partial t} \\ \frac{\partial^{2} \phi}{\partial t^{2}} = 2\tan{\theta}\frac{\partial \phi}{∂ t}\frac{\partial \theta}{\partial t} \end{cases}$$

Решение

Это система нелинейных ОДУ. Ауч. Плохо это тем, что подобные уравнения далеко не всегда вообще имеют аналитические решения. Но как увидим далее итоговые выражения получились вполне приятными(в том смысле, что там только элементарные функции. Работать с этим всё равно невозможно). Однако заметим, что второе уравнение представляет собой уравнение с разделяющимися переменными. После интегрирования получим

$$\frac{\partial \phi}{∂ t} = \frac{C _1}{\cos^2{\theta}}$$

А дальше, заметим что$\times 2$, в первом уравнении содержится лишь производная $\phi$, а значит мы свели задачу к уравнению чисто на функцию $\theta$

$$\frac{\partial^{2} \theta}{\partial t^{2}} = \sin{\theta}\cos{\theta}\frac{C^2 _1}{\cos^4{\theta}}$$

Дальше, воспользуемся крайне часто встречающимся в физике приёмом. В уравнении, где вторая производная $\theta$ равна выражению только от $\theta$, мы можем домножить обе части на $\frac{\partial \theta}{\partial t}$ и проинтегрировать. Тогда мы понизим порядок уравнения. И уже разделим переменные и вновь проинтегрируем. Эти преобразования оставлены читателю в качестве нетрудного упражнения. $$\frac{\arctan{\frac{(\sqrt{C _2} \sin{\theta})}{\sqrt{C^2 _1 + C _2 \cos^2{\theta}}}} \sqrt{C^2 _1 + C _2 \cos^2{\theta}}}{\sqrt{C _2} \cos{\theta} \sqrt{C _2 + \frac{C^2 _1}{\cos^2{\theta}}}} = t + C _3 = \frac{\arctan{\frac{(\sqrt{C _2} \sin{\theta})}{\sqrt{C^2 _1 + C _2 \cos^2{\theta}}}}}{\sqrt{C _2}}$$

Далее выразим $\theta$ через t

$$\frac{(\sqrt{C _2} \sin{\theta})}{\sqrt{C^2 _1 + C _2 \cos^2{\theta}}} = \tan{(\sqrt{C _2} t + \sqrt{C _2} C _3)}$$ $$\frac{C _2 \sin^2{\theta}}{C^2 _1 + C _2 \cos^2{\theta}} = \tan^2{(\sqrt{C _2} t + \sqrt{C _2} C _3)} $$ $$C _2 - C _2 \cos^2{\theta} = \tan^2{(\sqrt{C _2} t + \sqrt{C _2} C _3)}C^2 _1 + \tan^2{(\sqrt{C _2} t + \sqrt{C _2} C _3)} C _2 \cos^2{\theta} $$ $$\cos^2{\theta} C _2(\tan^2{(\sqrt{C _2} t + \sqrt{C _2} C _3)} +1) = C _2 - C^2 _1\tan^2{(\sqrt{C _2} t + \sqrt{C _2} C _3)}$$ $$\cos^2{\theta} = \frac{C _2 - C^2 _1\tan^2{(\sqrt{C _2} t + \sqrt{C _2} C _3)}}{C _2(\tan^2{(\sqrt{C _2} t + \sqrt{C _2} C _3)} +1)}$$ $$\cos^2{\theta} = \cos^2{(\sqrt{C _2} t + \sqrt{C _2} C _3)} - \frac{C^2 _1}{C _2} \sin^2{(\sqrt{C _2} t + \sqrt{C _2} C _3)}$$

Последнее выражение is a surprise tool, that will help us later

$$\theta = \arccos{\sqrt{\cos^2{(\sqrt{C _2} t + \sqrt{C _2} C _3)} - \frac{C^2 _1}{C _2} \sin^2{(\sqrt{C _2} t + \sqrt{C _2} C _3)}}} + 2\pi k, k\in\Z$$

Вот мы и получили решение для $\theta$. Благодаря выражению перед ним и результату самого первого разделения переменных получим уравнение на $\phi$

$$\frac{\partial \phi}{∂ t} = \frac{C _1}{\cos^2{\theta}} = \frac{C _1}{\cos^2{(\sqrt{C _2} t + \sqrt{C _2} C _3)} - \frac{C^2 _1}{C _2} \sin^2{(\sqrt{C _2} t + \sqrt{C _2} C _3)}}$$

Заметим, что$\times 3$ тут нигде не встречается $\phi$. А значит можно просто проинтегрировать обе части и получить решение. Это тоже оставлю в качестве упражнения

$$\phi = \tanh^{-1}{\left(\frac{C _1 \tan{(\sqrt{C _2} t + \sqrt{C _2} C _3)}}{\sqrt{C _2}}\right)} + C _4$$

И вот. Конец пути. Мы получили искомое решение. Но что же это за кривые? Каждому советскому пятикласснику только от одного вида понятно, что это уравнение больших кругов. Т.е. окружностей на сфере, у которых собственный центр с центром сферы совпадает.
